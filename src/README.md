# ShortMan Dictionary

> The internet slang dictionary for the chronically online.

A humorous, Gen-Z internet slang dictionary inspired by the layout and utility of the Longman Dictionary — but actually useful in 2025.

## Features

- **A–Z Navigation** — Browse slang by letter with an interactive sidebar
- **Search** — Real-time search with suggestions
- **Categories** — Gaming, TikTok, Reddit, Meme Terms
- **Brainrot Mode** — Toggle for maximum internet brain energy
- **Favorites** — Save your favorite entries locally
- **Community Words** — Submit and upvote your own slang
- **Dark Mode** — Easy on the eyes at 2AM
- **Responsive** — Works on mobile and desktop

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Base44 BaaS (database, auth)
- shadcn/ui components

## Getting Started

```bash
npm install
npm run dev
```

## Deploy

This project deploys automatically to GitHub Pages on every push to `main` via GitHub Actions.

Set the following repository secret:
- `VITE_BASE44_APP_ID` — your Base44 app ID
